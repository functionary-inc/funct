title: "webhook"

<Note>
  If you haven't learned about actions checkout the [actions page](/actions).
</Note>

Sends a http request to the a specified url with attributes.

<ParamField body="type" type="string = 'webhook'" required>
  Must equal the string `webhook`.
</ParamField>

<ParamField body="args" type="object" required>
  <Expandable title="properties" defaultOpen={true}>

    <ResponseField name="method" type="string = 'GET' or 'POST'" required>
      http method of the request.  Must be caps.  POST and GET are supported.
    </ResponseField>

    <ResponseField name="url" type="templatable string" required>
      url of the to make the request too.
    </ResponseField>

    <ResponseField name="headers" type="object">
      An object of the headers to send with the request.  The values of the headers will be templated with liquid.
    </ResponseField>

    <ResponseField name="body" type="object">
      An object of the body to send with the request. The values of the headers will be templated with liquid.

      <Info>Only Available for POST requests</Info>
    </ResponseField>

  </Expandable>
</ParamField>

```json
{
  "type": "webhook",
  "args": {
    "method": "POST",
    "url": "https://someService.com/{{modelProperties.serviceId}}",
    "headers": {
      "Authorization": "Bearer {{modelProperties.apiKey}}"
    },
    "body": {
      "taskCount": "{{eventProperties.taskCount}}"
    }
  }
}
```

<Tip>
  See [action](/actions) for all the accessible properties for templating
</Tip>
